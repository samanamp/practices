▶ Launch JupyterLite Launch Binder Open in Colab

JupyterLite runs fully in your browser — no install, numpy/pandas/matplotlib preloaded. The concurrency track needs real OS threads, so launch it on Binder or Colab. View source on GitHub →

Edits in JupyterLite are saved in your browser's local storage — use File → Download to keep a changed notebook. Binder sessions are temporary.  ·  samanamp/practices
